# Литература.Гид школьника

Мобильный гид для подростков 13–15 лет: сначала человек и контекст эпохи, затем классический текст. Первый маршрут — Лев Толстой, «После бала».

## Гипотеза MVP

Если перед чтением дать короткий и проверенный контекст, ребёнок откроет произведение и начнёт читать. Главное событие: `reading_started_confirmed`.

## Запуск

```bash
npm install
npm run dev
```

Проверки:

```bash
npm run validate:content
npm run typecheck
npm run build
```

## Структура

- `app/` — страницы и маршруты.
- `components/` — интерактивные элементы.
- `content/` — факты и локализованные тексты.
- `docs/` — продуктовые, редакционные и локализационные правила.
- `scripts/validate-content.mjs` — проверка обязательных блоков и источников.

## Редакционный цикл

Google Sheets → черновик в Google Docs → фактчекинг → структурированный контент в GitHub → автоматическая проверка → публикация.

# Продуктовые принципы

1. Не энциклопедия, а вход в текст через человека.
2. Каждый биографический факт обязан помогать прочитать конкретное произведение.
3. Контекст не пересказывает книгу и не выдаёт готовую мораль.
4. Пользователь — подросток; учитель и родитель помогают ему найти продукт.
5. Главный результат — ребёнок открыл произведение и начал читать.
6. Опубликованный контент в GitHub — единственный источник истины.
7. Красивый факт без проверяемого источника не публикуется.
8. Сначала одна работающая модель, потом каталог.

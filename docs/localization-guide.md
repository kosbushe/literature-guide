# Локализация

Проект готовится для `ru`, `en`, `es`, `fr`, но MVP публикуется по-русски.

- Факты, даты и связи лежат в `core.json`.
- Тексты интерфейса и объяснения лежат в `locales/<locale>.json`.
- ID сущностей и блоков не переводятся.
- Slug может быть отдельным для каждого языка.
- Аналогии делятся на `universal` и `locale_specific`.
- Изображения не содержат вшитого текста.
- Переводчик адаптирует культурный мост и школьную программу, а не только слова.
